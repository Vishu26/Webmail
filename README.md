# Webmail

Client for Mail with beautiful UI.

# Required Libraries
1. PyQt5<br />
2. Selenium<br />
3. RoboBrowser<br />

# Usage

If you have only Python3 installed:<br />
`python scraping.py`<br /><br />
If you have both Python3 and Python2 installed:<br />
`python3 scraping.py`
